═══════════════════════════════════════════════════════════════
✏️ COMMENT PERSONNALISER
═══════════════════════════════════════════════════════════════

Ouvrez le fichier "index.html" avec un éditeur de texte 
(Bloc-notes, Notepad++, VS Code, etc.) et modifiez :

📌 INFORMATIONS PERSONNELLES :
   - Ligne ~30-40 : Nom "KARA" dans le logo
   - Ligne ~75-80 : Titre et tagline du hero
   - Ligne ~100-140 : Description "À Propos"
   - Ligne ~145-175 : Détails personnels (nom, localisation, email)
   - Ligne ~440-460 : Coordonnées dans la section contact

📌 COMPÉTENCES :
   - Ligne ~180-250 : Barres de progression des compétences
   - Modifiez les pourcentages dans "style="width: XX%""

📌 PROJETS :
   - Ligne ~270-400 : Cartes de projets
   - Modifiez les titres, descriptions, technologies
   - Remplacez les images dans le dossier /images/

📌 LIENS SOCIAUX :
   - Ligne ~465-485 : URLs GitHub, LinkedIn, Twitter


═══════════════════════════════════════════════════════════════
📄 AJOUTER DES DOCUMENTS
═══════════════════════════════════════════════════════════════

Dans la section Portfolio, cliquez sur "Ajouter un document"
pour télécharger vos PDF et images.

Note : Les documents sont stockés temporairement dans le 
navigateur. Ils seront perdus si vous rafraîchissez la page.


═══════════════════════════════════════════════════════════════
🎨 CHANGER LE LOGO
═══════════════════════════════════════════════════════════════

1. Remplacez le fichier "images/logo.png" par votre logo
2. Taille recommandée : 512x512 pixels ou plus
3. Format : PNG (avec ou sans transparence)


═══════════════════════════════════════════════════════════════
🖼️ CHANGER LES IMAGES DE PROJETS
═══════════════════════════════════════════════════════════════

1. Remplacez les fichiers "images/project-*.png"
2. Taille recommandée : 1024x1024 pixels
3. Formats acceptés : PNG, JPG, WebP


═══════════════════════════════════════════════════════════════
🌐 METTRE EN LIGNE (OPTIONNEL)
═══════════════════════════════════════════════════════════════

Pour mettre votre portfolio en ligne :

1. GitHub Pages (gratuit) :
   - Créez un compte sur github.com
   - Créez un nouveau repository
   - Uploadez tous les fichiers
   - Activez GitHub Pages dans les settings

2. Netlify (gratuit) :
   - Allez sur netlify.com
   - Glissez-déposez le dossier
   - Votre site est en ligne !

3. Autres options :
   - Vercel (vercel.com)
   - Firebase Hosting
   - Votre propre hébergeur
